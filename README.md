# Twatch

Twatc is a static landing page for smartwatches. It is built with vanilla CSS and uses flexbox for the layout and it also uses jquery for basic interactions like a smooth scroll, slider, etc.

## Setup Step

1. Clone repo
2. `npm install`
3. `npm start`
